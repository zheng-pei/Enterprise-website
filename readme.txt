this is a chinese and english enterprise web ,I hope more persons can see these files .

访问的路径：前台：localhost://enterprise-web/public_html/index.php
            后台：localhost://enterprise-web/public_html/skadmin.php

数据库文件是sha.sql文件

文件的目录：
          Tpcore---------thinkphp3.2.3的核心文件

          WEBSITE--------前台和后台的代码
                 --------- WEBSITE/#WEB#/Common/Conf 里面db.php是数据库的配置文件

          public_html----static 静态文件
                     ----Uploads 上传图片的文件
                     ----index.php 入口文件
                     ----skadmin.php 后台入口的文件


默认的数据库名是sha.sql ，用户名是root，数据库密码为空
